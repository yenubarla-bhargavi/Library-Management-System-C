# Library Management System in C

## Overview

This project is a menu-driven Library Management System developed using the C programming language. It allows users to add books, search books, issue books, return books, display book records, and delete books. The system uses file handling to store and manage book information permanently.

---

## Features

- Add New Books
- Display All Books
- Search Book by ID
- Issue Book
- Return Book
- Delete Book
- Persistent Data Storage using Files
- Menu-Driven User Interface

---

## Technologies Used

- C Programming
- Structures
- Functions
- File Handling
- Binary Files
- CRUD Operations

---

## Concepts Covered

- Structures
- File Handling
- Record Management
- Search Operations
- Data Validation
- CRUD Operations
- Modular Programming

---

## Project Objective

To develop a simple library management application that demonstrates core C programming concepts and file-based record management while simulating real-world library operations.

---

## Project Workflow

1. Add New Book
2. Store Book Details in File
3. Search Book by ID
4. Issue Book
5. Return Book
6. Delete Book
7. Display All Books
